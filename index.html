Bahasa Pemrograman

Bahasa pemrograman adalah alat yang membantu programmer menulis instruksi a gar komputer dapat menjalankan tugas tertentu. Alasannya, komputer hanya memahami sistem bilangan biner, yaitu “1” dan “0”. Untuk memungkinkan programmer memberikan perintah yang dimengerti logika komputer, dibutuhkan bahasa pemrograman sebagai jembatan komunikasi. Itulah tujuan utama dibuatnya bahasa pemrograman.

Apa Fungsi Bahasa Pemrograman?

Fungsi bahasa dalam kehidupan manusia adalah sebagai alat komunikasi yang dapat dipahami. Begitu pula dengan bahasa pemrograman, yang berfungsi sebagai penghubung komunikasi antara komputer dan manusia (programmer). Namun, bahasa manusia dengan komputer tentu berbeda. Hasil dari bahasa pemrograman adalah sistem operasi, aplikasi desktop, aplikasi mobile, website, hingga perangkat teknologi yang kita gunakan sehari-hari seperti ponsel.

Jenis-jenis Bahasa Pemrograman

  1. Low-level language atau bahasa mesin

    2. Bahasa pemrograman tingkat rendah adalah bahasa yang dekat dengan kode mesin atau bahasa asli komputer. Bahasa ini sulit dibaca manusia karena erat dengan penulisan kode mesin dan berhubungan langsung dengan komponen hardware komputer. Instruksi dari bahasa tingkat rendah bersifat kontrol langsung, sehingga masih diperlukan untuk fungsi yang terkait dengan struktur fisik komputer.

  2. Assembly language atau bahasa rakitan

    3. Assembly language adalah bahasa pemrograman tingkat rendah yang dirancang agar dapat dipahami manusia. Meskipun masih termasuk low-level language, bahasa ini lebih mudah dipahami daripada bahasa mesin. Bahasa ini digunakan sebagai penghubung antara bahasa pemrograman tingkat tinggi (seperti Python atau SQL) dengan hardware dasar komputer.

  3. High-level language atau bahasa tingkat tinggi

    4. Bahasa pemrograman tingkat tinggi lebih dekat dengan cara manusia berkomunikasi. Dalam high-level language, kata-kata yang digunakan mencakup istilah yang mudah dipahami seperti order, request, run, dan object. Bahasa ini lebih manusiawi dan mudah dipelajari oleh programmer.

Dart

Dart adalah bahasa pemrograman yang dikembangkan oleh Google untuk pengembangan aplikasi frontend, terutama yang berjalan di Flutter untuk aplikasi mobile. Dart dikenal dengan kemampuannya yang cepat dan sintaks yang mirip dengan JavaScript.

Fitur Utama Dart

  • Optimasi: Memiliki performa tinggi dan cepat untuk aplikasi mobile.
  • Null Safety: Fitur untuk mengurangi kesalahan yang berhubungan dengan nilai null.

Contoh Kode Dart

        void main() {
            print('Hello, World!');
        }
Output:
Hello, World!

C++

C++ adalah bahasa pemrograman berperforma tinggi yang sering digunakan untuk pengembangan perangkat lunak dan aplikasi yang membutuhkan efisiensi tinggi, seperti game dan sistem operasi.

Fitur Utama C++

  • Performa Tinggi: Memberikan kontrol penuh terhadap hardware.
  • Dukungan Paradigma Beragam: Mendukung pemrograman prosedural, berorientasi objek, dan generik.

Contoh Kode C++

        #include 
        using namespace std;
        
        int main() {
            cout << "Hello, World!" << endl;
            return 0;
        }
Output:
Hello, World!

Java

Java adalah bahasa pemrograman berorientasi objek yang dirancang untuk memiliki portabilitas tinggi. Pertama kali dirilis oleh Sun Microsystems pada tahun 1995, Java sangat populer dalam pengembangan aplikasi perusahaan dan aplikasi Android.

Fitur Utama Java

  • Portabilitas: Kode Java dapat berjalan di mana saja melalui Java Virtual Machine (JVM).
  • Keamanan: Java memiliki fitur keamanan bawaan yang kuat.

Contoh Kode Java

            public class HelloWorld {
                public static void main(String[] args) {
                    System.out.println("Hello, World!");
                }
            }
Output:
Hello, World!

Ruby

Ruby adalah bahasa pemrograman dinamis dan berorientasi objek yang terkenal dengan sintaksnya yang mudah dipahami. Ruby sering digunakan dalam pengembangan aplikasi web menggunakan framework Ruby on Rails.

Fitur Utama Ruby

  • Dinamis dan Fleksibel: Mendukung metaprogramming dan sangat dinamis.
  • Framework Ruby on Rails: Menyediakan cara cepat untuk membangun aplikasi web menggunakan pola MVC.

Contoh Kode Ruby

            def hello_world
                puts "Hello, World!"
            end
            
            hello_world
Output:
Hello, World!

Swift

Swift adalah bahasa pemrograman yang dikembangkan oleh Apple untuk menggantikan Objective-C. Digunakan untuk membangun aplikasi iOS dan macOS, Swift dirancang untuk efisiensi dan keamanan.

Fitur Utama Swift

  • Efisiensi dan Performa Tinggi: Swift dirancang agar cepat dan efisien.
  • Keamanan: Mengurangi kesalahan dengan pengelolaan memori otomatis dan tipe yang aman.

Contoh Kode Swift

            import SwiftUI
            
            struct ContentView: View {
                var body: some View {
                    Text("Hello, World!")
                }
            }
Output:
Hello, World!
